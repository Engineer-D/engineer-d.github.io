## Overview

An edge-deployed computer vision system for detecting farm intrusions, using a lightweight Faster R-CNN with MobileNet backbone. Achieves 97-99% accuracy while running on ESP32 cameras and Raspberry Pi hardware.

## Motivation

Small-scale farmers in Nigeria face significant losses from animal and human intrusions. This system provides affordable, automated surveillance using commodity hardware.
