## Overview

Kiki is a telephony-integrated voice AI health companion built at Fertitude, supporting 5 Nigerian languages — Yoruba, Igbo, Hausa, Pidgin, and English. It provides personalized health support through natural voice conversations.

## Key Features

- **Multilingual Voice AI**: Real-time speech-to-text and text-to-speech pipelines supporting 5 languages with natural conversational flow
- **Telephony Integration**: Users interact through phone calls, making the service accessible without smartphones or internet
- **Mood Prediction**: Cycle-based mood prediction using contextual signals and conversation patterns
- **Context-Aware Notifications**: Personalized health reminders and check-ins based on user history and predicted needs
- **Content Recommendations**: Intelligent matching of users with health experts and community resources

## Technical Stack

- Real-time speech-to-text pipeline with language detection
- Text-to-speech synthesis with natural prosody per language
- Conversational AI with multi-turn context management
- Mood prediction model with temporal features
- Telephony infrastructure integration

## Challenges

Building multilingual voice AI for low-resource Nigerian languages required creative approaches to data collection, model adaptation, and evaluation — standard benchmarks don't cover these languages well.
